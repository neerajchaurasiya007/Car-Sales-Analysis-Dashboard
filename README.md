# Car-Sales-Analysis-Dashboard
This Power BI dashboard provides a comprehensive analysis of Year-to-Date (YTD) and Month-to-Date (MTD) car sales performance. It tracks critical KPIs, regional distribution, and manufacturer performance to enable data-driven decision-making for automotive retail stakeholders.
